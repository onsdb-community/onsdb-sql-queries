# Nedap OnsDB - SQL Queries

Deze repository bevat **herbruikbare SQL-queries voor Nedap OnsDB**, gedeeld door en voor zorgorganisaties die met Nedap ONS werken.

Het doel is om kennis te delen en dubbel werk te voorkomen.


## Structuur

```
queries/        # De SQL-querybibliotheek
templates/      # Query-template voor nieuwe bijdragen
```

## Bijdragen

Bijdragen zijn welkom 🎉 Heb je ook een query die je wilt toevoegen? Mail dan naar [c.vanson@odion.nl](c.vanson@odion.nl) om toegevoegd te worden als collaborator aan deze repository.

Nieuwe queries en wijzigingen worden altijd gedaan via een Pull Request (PR). Dit zorgt ervoor dat wijzigingen even kunnen worden bekeken voordat ze worden toegevoegd, en voorkomt dat iemand per ongeluk iets kapot maakt.

Niet bekend met `git`? GitHub begeleidt dit automatisch,
ook als je alleen in de browser werkt. Dat werkt zo:

**Werkwijze**
1. Ga naar de map `queries/`
2. Klik op **Add file → Create new file** (nieuwe query)  
   of open een bestaande query en klik op ✏️ **Edit**
3. Gebruik bij voorkeur het template in `templates/query-template.sql`
4. Klik op **Propose changes** en maak de Pull Request aan

**Richtlijnen**
- Houd de documentatie **kort en functioneel**
- Voeg **geen cliëntdata, exports of screenshots** toe
- Gebruik placeholders waar nodig
